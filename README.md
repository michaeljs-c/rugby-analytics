# Rugby ETL

This repository downloads, processes and stores rugby data from ESPN. 

The project is runnable locally and on AWS.

## Tables
1. Game stats
2. Player stats
3. Match details

## Project Components

1. Ingest (download)
2. Transform/model
3. Load/Store
4. Daily ETL scheduling
5. Create API
6. Create dashboard
